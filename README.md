## 散列表
### Description
学校想要开发一个的学号存储系统，支持以下两种操作：

1. I x，插入一个学号`x`

2. Q x，查询学号`x`是否在系统中

现在要进行N次操作，对于每个查询输出对应的结果。


### Input 
第一行包含整数N，表示操作数量。
操作数量范围：1＜ N ＜10^5
后N行，每行包含一个操作指令，操作指令为`I x`，`Q x`中的一种。 学号x的范围：-10^9 ＜ x ＜ 10^9


### Output 
对于每个询问指令`Q x`，输出一个询问结果，如果x在集合中出现过，则输出Yes，否则输出No。

每个结果占一行。(每次输出后加入回车)
### Sample Input 1 

>5
>I 1
>I 2
>I 3
>Q 2
>Q 5

### Sample Output 1 

> Yes
> No
